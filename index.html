Welcome to My Basic Website

I created this website for the virtualization solutions engineer role at Cloudflare

Making this small project step-by-step

Step 1 - Registering a domain

  • As Freenom was not working correctly, I used the Namecheap tool
  • Registered the domain "cloudflrtest.shop"

Step 2 - Adding the domain to Cloudflare

  • Created a free tier account on Cloudflare
  • Added the website domain on the "Website" menu on the main dashboard

Step 3 - Changing name servers

  • On Namecheap, I changed the two domain name entries to the ones referred to on Cloudflare
  • I waited to receive an email notification saying the domain was active on Cloudflare

Step 4 - Create a simple website and host it online

  • Created the website using sublime text
  • Uploaded the index.html file on an AWS S3 Bucket
  • Created an AWS Cloudfront distribution for the bucket
  • Protected the communication Between customers, Cloudflare, and AWS hosting through TLS on the Cloudflare platform
  • Made a CNAME entry on Cloudflare DNS to point the domain cloudflrtest.shop to the distribution domain of AWS Cloudfront
